Antenna Rotor Controller
------------------------

Prerequisites:

    $ sudo apt install libgtk-3-dev
    $ sudo apt install glade

Download:

    git clone https://github.com/wryan67/rotor.git
    
Compile:

    $ make
    
Run:

    $ ./rotor
